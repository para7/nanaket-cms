# Product Overview

Headless CMSプラットフォーム「Nanaket CMS」- コンテンツ管理とAPIベースの配信を提供

## Core Capabilities

- **RESTful API**: `/api/v1/`エンドポイントによるコンテンツアクセス
- **User Management**: ユーザーCRUD操作（認証・認可は今後実装予定）
- **Database-Driven**: PostgreSQL永続化によるスケーラブルなデータ管理
- **Health Monitoring**: システム健全性チェックとDB接続状態の可視化

## Target Use Cases

- コンテンツ管理が必要なWebアプリケーション/モバイルアプリ
- マルチチャネル配信（Web、アプリ、IoT）向けコンテンツソース
- API経由でコンテンツを取得するフロントエンド開発

## Value Proposition

- **Headless Architecture**: フロントエンドから独立した柔軟なコンテンツ配信
- **Go Performance**: 高速で同時実行性の高いREST API
- **Type-Safe Database**: sqlcによるコンパイル時型チェック付きクエリ
